CSS overflow Property Examples

overflow: visible (default)

This is some long content that will overflow outside the box. The overflow is visible so you can see it extending beyond the container!

overflow: hidden

This is some long content that will overflow outside the box. The overflow is hidden and you will only see what fits in the box.

overflow: scroll

This is some long content that will overflow outside the box. Overflow is scroll, so scrollbars are always visible, even if you don't need them.

overflow: auto

This is some long content that will overflow outside the box. Overflow is auto: scrollbars appear only when needed.

overflow: clip

This is some long content that will overflow outside the box. Overflow is clip: content is simply cut off, with no scrollbars and no visible overflow.
